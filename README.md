# SETOR 7 - Sistema de Controle de Unidades

Este é um Dashboard interativo que eu desenvolvi no intuito apenas de aprendizagem, com temática em Pixel Art.

##  Tecnologias Utilizadas
* **HTML5** (Estrutura semântica)
* **CSS3** (Estilização retro/pixel art, Scanlines e animações)
* **JavaScript** (Lógica de estados e manipulação de DOM)
* **LocalStorage** (Persistência de dados do usuário)

##  Funcionalidades
- **Recrutamento/Input:** Validação de nomes (apenas letras, máx 12 caracteres).
- **Designação Random:** O sistema sorteia um treinamento específico para cada soldado.
- **Persistência:** Suas unidades ficam salvas mesmo após fechar o navegador.
